# FXTSX-HTML5

FXTSX를 이용하여 HTML5를 통해 구현한 컴포넌트입니다. 마크업과 기능만 구현되어 있으며 스타일 구현은 각 프로젝트에 위임함으로써 각 프로젝트간의 컴포넌트 호환성을 보장합니다.

## 구성
- Sectioning
    - Heading
    - Section
    - Article
    - Nav
    - Aside
- FormAssociated
    - Form
    - Fieldset
    - TagList
    - inputFieldWrapper
    - TextField
    - CheckField
    - FileField
    - TextArea
- UI
    - Check
    - Button
    - Dialog
    - Figure
    - Img
    - Tag
    - DropDown
    - Selectbox
