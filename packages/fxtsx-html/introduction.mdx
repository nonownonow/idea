# FXTSX-HTML5

FXTSX를 이용하여 HTML5를 통해 구현한 컴포넌트입니다. 마크업만 구성되어 있으며 스타일은 각 프로젝트에 위임합니다. FXTSX-HTML5를 통하여 구현된 컴포넌트는 동일한 인터페이스로 구

## 구성
    - Sectioning
        - Heading
        - Section
        - Article
        - Nav
        - Aside
    - FormAssociated
        - Form
        - Fieldset
        - TagList
        - inputFieldWrapper
        - TextField
        - CheckField
        - FileField
        - TextArea
    - UI
        - Check
        - Button
        - Dialog
        - Figure
        - Img
        - Tag
        - DropDown
        - Selectbox
